Copies a surface, or a subpart, to another one. The source and destination area of the copy (either the entire surfaces, or subparts) must have the same size (before clipping). Both the source and destination surfaces must be $\ast$locked$\ast$ by \hyperlink{hw__interface_8h_a1b9b22f29bb9de0b39ea904fd66251be}{hw\_\-surface\_\-lock}. 
\begin{DoxyParams}{Parameters}
\item[{\em destination}]The surface on which to copy pixels from the source surface. \item[{\em dst\_\-rect}]If NULL, the entire destination surface is used. If not NULL, defines the rectangle on the destination surface where to copy the pixels. \item[{\em source}]The surface from which to copy pixels. \item[{\em src\_\-rect}]If NULL, the entire source surface is used. If not NULL, defines the rectangle on the source surface from which to copy the pixels. \item[{\em alpha}]If true, the final pixels are a combination of source and destination pixels weighted by the source alpha channel. The transparency of the final pixels is set to opaque. If false, the final pixels are an exact copy of the source pixels, including the alpha channel.\end{DoxyParams}
\begin{DoxyReturn}{Returns}
Returns 0 on success, 1 on failure (different ROI size). 
\end{DoxyReturn}
